AI Trip Planner ✈️

Project Overview
AI Trip Planner is an intelligent travel planning system that generates personalized travel itineraries based on user preferences, and interests.

Objectives
- Provide smart destination recommendations
- Generate customized travel itineraries
- Reduce manual travel planning effort
- Improve user experience using AI

Technologies Used
- Python
- openAI API
- Flask 
- SQLite / MySQL 

Features
- User input-based trip customization
- AI-generated day-wise itinerary
- Destination recommendations
- Travel history management
- Simple and interactive interface
- Restaurant recommendations
